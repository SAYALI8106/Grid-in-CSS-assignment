In CSS Grid, `justify-items` and `justify-self` are properties used to align grid items along the inline (row) axis within their grid areas. Here's a detailed explanation with code examples: ### `justify-items` The `justify-items` property aligns all the grid items within their respective grid areas along the inline (row) axis. #### Values: - `start`: Aligns the items at the start of the grid area. - `end`: Aligns the items at the end of the grid area. - `center`: Aligns the items at the center of the grid area. - `stretch`: Stretches the items to fill the grid area (default). #### Example: ```html
1
2
3
4
5
6
``` In this example, all grid items are centered within their respective grid areas. ### `justify-self` The `justify-self` property aligns a single grid item within its grid area along the inline (row) axis. It overrides the `justify-items` value for the specific item. #### Values: - `start`: Aligns the item at the start of the grid area. - `end`: Aligns the item at the end of the grid area. - `center`: Aligns the item at the center of the grid area. - `stretch`: Stretches the item to fill the grid area (default). #### Example: ```html
1
2
3
4
5
6
``` In this example, all grid items are aligned to the start of their grid areas, except for the second item, which is aligned to the end of its grid area due to the `justify-self: end;` property.
